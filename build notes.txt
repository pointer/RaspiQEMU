To make this seamless for anyone using your repository, you can automate this mapping so you don't have to type it every time.
Option A: Create a .gdbinit file in the VM

Create a file named .gdbinit in the debian user's home directory inside the VM:
Bash

# Inside the VM
cat <<EOF > ~/.gdbinit
set substitute-path /home/kateb/qemu/shared_project /mnt/shared_project
set pagination off
EOF
